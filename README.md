# LegalLens AI

A complete LegalLens AI application scaffold built for legal document analysis.

## What is included

- React + Vite frontend
- Express + TypeScript backend
- Local LegalLens analysis engine with document profiling, risk detection, clause scanning, and Q&A support

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the app:
   ```bash
   npm run dev
   ```

3. Open the frontend at:
   ```text
   http://localhost:5173
   ```

## How it works

- Paste or upload a legal document as text
- Click **Analyze Document** to generate a structured report
- Ask a follow-up question using the built-in Q&A engine

## Notes

- The current implementation is a local analysis engine using heuristic rules.
- This is a complete app scaffold ready for additional AI or document parsing integration.
